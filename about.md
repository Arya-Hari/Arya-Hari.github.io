## Industry Experience

**Microsoft India (R&D)** — Software Engineering Intern *(May–June 2025)*
I worked on frontend features for Microsoft Teams Search, improving query responsiveness and click-through flows across 170M+ monthly user interactions. This involved optimizing React/TypeScript components to improve task completion rates at scale.

**HPCC Systems, LexisNexis Risk Solutions** — Project Intern *(December 2023–October 2024)*
I built an NLP-driven legal research assistant that extracts key phrases from case abstracts and retrieves relevant precedents using Named Entity Recognition and spaCy — achieving 15% higher precision over baseline techniques. The retrieval and ranking system was deployed on HPCC Systems using ECL, achieving sub-second responses on distributed data. This work won **Best Poster – Use Case** at the 2024 HPCC Systems Technical Poster Competition.
