## Published

### Bridging Language Barriers in Coding: An NLP-Powered Tool for Programming Education in Regional Languages
**Authors:** Sooda et al.
**Venue:** IEEE Access, 2025
Developed a Kannada-language coding tutor using a LoRA fine-tuned CodeT5 Base model, achieving 3× improvement in code-generation accuracy over baseline. Evaluated with 50+ student test cases in a rural pilot program.
[DOI: 10.1109/ACCESS.2025.3587056](https://doi.org/10.1109/ACCESS.2025.3587056)
